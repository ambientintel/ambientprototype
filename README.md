# ambientprototype

Prototype UI for Ambient Intelligence. Not for production use.

## Local dev

```bash
npm install
npm run dev
```

Opens at http://localhost:3000

Pushes to `main` auto-deploy via Vercel.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS
